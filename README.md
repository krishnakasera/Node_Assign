# Node.js Server Assignment

## Description

This project demonstrates a basic Node.js server using the http module.

## Features

* Serves static HTML file
* Logs requests with timestamp
* Uses nodemon for auto reload

## How to Run

1. Install dependencies:
   npm install

2. Start server:
   npm start

3. For development:
   npm run dev

## Output

* Runs on http://localhost:3000
* Displays HTML page
* Logs request details in console
